## 📗 Getting Started (For Humans)

You don't need to be a coder to use this vault! If you want to explore the history and culture of Mizoram in a beautiful, interactive way, follow these steps:

### 1. Download Obsidian
Go to [Obsidian.md](https://obsidian.md/) and install the app. It's a free, private "Second Brain" tool that makes this repository come alive.

### 2. Get the Vault
-   Click the green **"Code"** button at the top of this page.
-   Select **"Download ZIP"**.
-   Unzip the folder onto your computer.

### 3. Open in Obsidian
-   Open the Obsidian app.
-   Select **"Open folder as vault"**.
-   Navigate to the unzipped folder and click **Open**.

> [!TIP]
> **Hit `Ctrl + G`** (Windows) or `Cmd + G` (Mac) once inside Obsidian to see the **Global Knowledge Graph**. It will show you a 3D map of how every Mizo concept is connected!
