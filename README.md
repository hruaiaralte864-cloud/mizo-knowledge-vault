# Mizo Knowledge Vault (Open Source)

A high-resolution "Digital Brain" of the Mizo people. This repository is an Obsidian-ready, LLM-optimized knowledge graph covering history, culture, science, and the 2026 digital frontier of Mizoram.

## 🌟 Features
- **360+ Deep Research Nodes**: From ancient metallurgy to aerospace.
- **800+ Dictionary entries**: Systematic A-Z Mizo-English concepts.
- **Semantic Net**: Every node is linked [[LikeThis]] to show the hidden connections in Mizo life.
- **2026 Ready**: Includes current data on Drone Logistics, Smart Cities, and Law evolutions.

## 📂 Structure
- `/Knowledge`: Categorized deep-dive articles.
- `/Dictionary`: Alphabetical terminology.
- `Master_Index.md`: The central hub for all links.

## 🚀 How to Use
1. **Obsidian**: Open this folder as a vault in Obsidian for the full graph experience.
2. **AI/LLMs**: Use these files as a high-fidelity context for RAG or AI training.
3. **Manual**: Browse the `/Knowledge` folders via GitHub.
