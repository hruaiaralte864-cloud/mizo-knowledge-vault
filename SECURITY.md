# Security Policy: Mizo Knowledge Vault

## Supported Versions
As this is a data-only repository composed of Markdown files, safety is focused on **Data Integrity**.

| Version | Supported |
| :--- | :--- |
| **v1.0 (Current)** | ✅ |

## Reporting a Vulnerability
If you discover a file that contains misinformation, harmful data, or sensitive PII (Personally Identifiable Information) that should not be public:

1.  **Do not** open a public issue.
2.  Please contact the maintainer at: `hruaiaralte864@gmail.com`
3.  We will respond within 48 hours to rectify the data or remove sensitive contents.

## Our Commitment
We are committed to maintaining a "Clean Data" environment. We scan all submissions to ensure no malicious scripts or non-text binaries are hidden within the vault.
